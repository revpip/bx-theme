# Business X-Ray Framework Theme

A premium WordPress block theme for the Business X-Ray ecosystem.

## Purpose

The theme provides the public website experience for Business X-Ray: a calm, premium, conversion-focused design system for business diagnostics, assessment landing pages, reports, case studies and client confidence-building content.

## Principles

- Theme handles presentation only.
- Business logic stays inside `bx-platform`.
- Native WordPress block theme foundations.
- Fast, accessible and mobile-first.
- Designed around clarity, confidence and calm.
